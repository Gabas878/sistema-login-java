# 🔐 Sistema de Login em Java

Este projeto é um sistema simples de autenticação desenvolvido em Java, com foco em prática de lógica de programação, orientação a objetos e validações de regras de negócio.

---

## 🚀 Funcionalidades

- Cadastro de usuário
- Validação de email duplicado
- Login com verificação de credenciais
- Tratamento de erros:
  - Usuário já cadastrado
  - Usuário não encontrado
  - Senha incorreta

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Listas (`ArrayList`)
- Estruturas de decisão (`if/else`)
- Boas práticas de organização de código

---

## 📂 Estrutura do Projeto

```
model/

service/
 └── SistemaLogin.java

 └── SistemaTeste.java
```
